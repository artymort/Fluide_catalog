# FLUIDE — заметки

- Делаем tablet-first электронный каталог парфюмерии.
- Создана стартовая страница: зацикленное видео, синее затемнение, белый логотип и кнопка перехода.
- Добавлена адаптация заставки для планшета, ПК и узкого вертикального экрана.
- Основной шрифт — Mazzard H; ожидаются файлы шрифта.
- Все тексты интерфейса используют начертание Regular (400).
- Проверено наличие всех подключенных файлов; визуальная проверка ожидает доступного окна браузера.
- Исправлены слои заставки: видео теперь находится перед фоном, а затемнение и интерфейс — поверх видео.
- Управление встроенным браузером подключено, но для проверки требуется открыть активное окно Browser/Preview в Codex.
- Проверен MP4: H.264, 1264×720. SVG оказался непрозрачным; для него включено наложение `multiply`, чтобы видео было видно.
- Добавлено обновление версии CSS в ссылке, чтобы браузер не показывал старый закэшированный стиль.
- Кнопка закреплена точно по центру экрана; положение логотипа рассчитывается отдельно.
- Получен визуальный референс каталога: светлый минимализм, фиолетовые акценты, крупные tablet-first элементы, карточная сетка и пошаговый подбор.
- Из брендбука взяты цвета: #434865, #F4F0E6, #9FB6DF, #EAB6A8, #ECB0B5.
- Создан фиксированный главный экран каталога: логотип, заголовок и семь разделов; прокрутка на планшете отключена.
- Кнопка на заставке подключена к главному меню `catalog.html`.
- Проект опубликован в публичном репозитории `artymort/Fluide_catalog` и развёрнут через GitHub Pages.
- Добавлен PWA-режим: манифест, Service Worker, иконки и полноэкранный запуск с главного экрана планшета.
- Следующий этап — наполнение карточек каталога продукцией.
